mllib.classification.LinearClassificationModel¶

class mllib.classification.LinearClassificationModel(weights, intercept)[source]¶

A private abstract class representing a multiclass classification model. The categories are represented by int values: 0, 1, 2, etc.

__init__(weights, intercept)[source]¶

Methods¶

__init__(weights, intercept)
clearThreshold() Clears the threshold so that predict will output raw prediction scores.
predict(test) Predict values for a single data point or an RDD of points using the model trained.
setThreshold(value) Sets the threshold that separates positive predictions from negative predictions.

Attributes¶

intercept Intercept computed for this model.
threshold Returns the threshold (if any) used for converting raw prediction scores into 0/1 predictions.
weights Weights computed for every feature.
