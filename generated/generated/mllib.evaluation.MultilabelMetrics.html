class mllib.evaluation.MultilabelMetrics(predictionAndLabels)[source]Â¶

Evaluator for multilabel classification.

Parameters:predictionAndLabels – an RDD of (predictions, labels) pairs, both are non-null Arrays, each with unique elements. 
>>> predictionAndLabels = sc.parallelize([([0.0, 1.0], [0.0, 2.0]), ([0.0, 2.0], [0.0, 1.0]),
...     ([], [0.0]), ([2.0], [2.0]), ([2.0, 0.0], [2.0, 0.0]),
...     ([0.0, 1.0, 2.0], [0.0, 1.0]), ([1.0], [1.0, 2.0])])
>>> metrics = MultilabelMetrics(predictionAndLabels)
>>> metrics.precision(0.0)
1.0
>>> metrics.recall(1.0)
0.66...
>>> metrics.f1Measure(2.0)
0.5
>>> metrics.precision()
0.66...
>>> metrics.recall()
0.64...
>>> metrics.f1Measure()
0.63...
>>> metrics.microPrecision
0.72...
>>> metrics.microRecall
0.66...
>>> metrics.microF1Measure
0.69...
>>> metrics.hammingLoss
0.33...
>>> metrics.subsetAccuracy
0.28...
>>> metrics.accuracy
0.54...
