3.6.2.18. pyspark.ml.linalg.Vectors¶

class pyspark.ml.linalg.Vectors[source]¶

Factory methods for working with vectors. Note that dense vectors are simply represented as NumPy array objects, so there is no need to covert them for use in MLlib. For sparse vectors, the factory methods in this class create an MLlib-compatible type, or users can pass in SciPy’s C{scipy.sparse} column vectors.

Methods

dense(*elements) Create a dense vector of 64-bit floats from a Python list or numbers.
norm(vector, p) Find norm of the given vector.
sparse(size, *args) Create a sparse vector, using either a dictionary, a list of (index, value) pairs, or two separate arrays of indices and values (sorted by index).
squared_distance(v1, v2) Squared distance between two vectors.
